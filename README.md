# Проекты, выполненные в рамках курса "Аналитик данных"
1. **project_analysis_data_fantasy** - проект спринта по анализу данных и решению ad hoc задач. 
2. **project_dashboard_ted** - проект спринта по визуализации данных с помощью Datalens и созданию дашбордов.
3. **project_analysis_data_real_estate** - итоговый проект первого модуля по анализу данных агентства недвижимости.
4. **project_videogame_market** - проект спринта по анализу данных рынка видеоигр.
