# Ключевые бизнес-метрики сервиса доставки еды

> Дашборд в DataLens для анализа клиентской базы сервиса «Всё.из.кафе» (г. Саранск, май–июнь 2021)

---

## Цель проекта
Разработать интерактивный дашборд для мониторинга ключевых метрик бизнеса, чтобы на основе данных принимать решения по удержанию клиентов и оптимизации маркетинговых затрат.

## Задачи
- Подготовить QL-запросы в DataLens для расчёта продуктовых метрик
- Создать визуализации, отражающие динамику и аномалии показателей
- Собрать единый дашборд для демонстрации стейкхолдерам
- Сформулировать практические выводы по периоду май–июнь 2021

## Анализируемые метрики
| Метрика | Описание | Бизнес-вопрос |
|---------|----------|---------------|
| **DAU** | Daily Active Users — уникальные пользователи за день | Растёт ли аудитория? |
| **Conversion Rate** | Доля пользователей, совершивших заказ | Насколько эффективна воронка? |
| **Средний чек** | Средняя сумма покупки на пользователя | Меняется ли ценность заказа? |
| **LTV** | Lifetime Value — совокупная ценность клиента за период | Окупаются ли затраты на привлечение? |
| **Retention Rate** | Доля пользователей, вернувшихся в продукт | Удается ли удерживать клиентов? |

## Источники данных
Проект основан на 5 таблицах:
- `analytics_events` — логи событий пользователей (просмотры, заказы, авторизации)
- `advertisement_budgets` — ежедневные рекламные затраты по каналам
- `partners` — справочник ресторанов-партнёров
- `dishes` — каталог блюд с атрибутами (острое, мясное, рыбное)
- `cities` — справочник городов присутствия

*Период анализа: 30.04.2021 – 02.07.2021, фокус на г. Саранск*
